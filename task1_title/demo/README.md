# Environments

```bash
pip install voila==0.2.4 voila-gridstack
```

# Run

`voila`를 입력하여 실행

```bash
voila inference.ipynb --template gridstack --VoilaConfiguration.resources='{"gridstack": {"show_handles": True}}'
```
